# 05_결과물 디렉터리 통합

## 변경 사항
- 루트에 흩어진 결과물을 `results/` 디렉터리로 통합:
  - `kf_hist_*` (6개), `kf_result_*` (7개), `kf_Comp/` — KFold 학습 기록/결과
  - `ELEC_*.png` (18개) — 클러스터링/모델 비교 시각화 이미지
  - `[BU] *` (2개) — 백업 결과
  - `[미사용] *` (6개) — 미사용 결과
- 빈 디렉터리 `WS_TS_5Energy_Clustering/` 삭제

## 이점
- 루트 디렉터리 38개 항목 → 7개로 축소
- 소스 코드와 실험 결과물의 명확한 분리
- 프로젝트 구조 파악이 즉시 가능
